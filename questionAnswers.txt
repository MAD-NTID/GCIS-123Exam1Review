abstract class
================
 - can have state
 - can be extended
 - can have constructor
 - abstract methods doesnt have body
 - only one abstract class

interface
===========
- cannot have state
 - can be implemented using the implemented keyword, you can extend to another interface
 - cant have constructor
 - can implement several interfaces
 - help achieve loosely coupling
